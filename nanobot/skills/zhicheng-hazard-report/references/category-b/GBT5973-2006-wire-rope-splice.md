# 钢丝绳用楔形接头 — 原文精要

- 源文件: B类规范（国家）\GBT5973-2006钢丝绳用楔形接头.pdf
- 性质: 中华人民共和国国家标准
- 编号: GB/T 5973-2006
- 适用范围: 起重机械、钢丝绳牵引设施中钢丝绳与楔形接头的连接
- 关联清单: references/CASES_XLSX.md → 起重机械、钢丝绳

---

## 核心要点

### 型式与规格
- 楔形接头按钢丝绳直径选用相应规格，不许混用
- 接头本体、楔套须有材质和规格标识

### 安装与使用
- 钢丝绳穿绕方向须正确，绳端应稳固卡紧
- 安装后须检查楔套与绳的贴合程度，确保可靠锁紧
- 使用中须定期检查绳在楔套内的位置，有窜动须重新卡紧

### 报废标准
- 楔套裂纹、变形、严重磨损须报废
- 钢丝绳在楔套内松动、绳股断裂须更换

## 重大隐患关联
- 使用非标或规格不匹配楔形接头 → 重大隐患
- 楔套有裂纹/变形继续使用 → 重大隐患
- 钢丝绳在楔套内松动未重新卡紧 → 重大隐患
