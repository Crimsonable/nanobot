# 四川省安全生产条例 — 原文精要

- 源文件: B类规范（国家）\四川省安全生产条例.docx
- 性质: 地方性法规
- 适用范围: 四川省行政区域内从事生产经营活动的单位的安全生产管理
- 关联清单: references/CASES_XLSX.md → 管理职责、风险管控、隐患排查、教育培训、应急救援

---

## 核心制度要点
- 安全责任体系：生产经营单位的主要负责人是安全生产第一责任人；建立全员安全生产责任制
- 风险分级管控与隐患排查治理：双重预防机制落实，定期排查、及时整改闭环
- 安全投入：保证安全生产条件所必需的资金投入，安全费用计提使用合规
- 教育培训：主要负责人、安全管理人员、特种作业人员、从业人员培训持证上岗
- 应急管理：编制应急预案并定期演练，配备应急救援器材
- 施工现场管理：高处、临边、交叉作业安全防护，动火/临时用电/起重等危险作业管理
- 相关方管理：承包租赁安全管理，统一协调管理，不得以包代管
- 监督检查：属地监管与行业监管相结合，重大隐患挂牌督办

## 重大隐患关联
- 未建立隐患排查制度、重大隐患未整改闭环 → 重大隐患
- 现场安全防护缺失、安全标志不足 → 严查严管项
- 相关方安全管理缺失、以包代管 → 重大隐患
