## 主要内容与强制性条文

### 3.1 建筑分类与耐火等级
- 民用建筑分类：一类高层、二类高层、单多层
- 厂房、仓库的耐火等级分一、二、三、四级
- 防火墙耐火极限不低于3h，楼板不低于1.5h

### 3.4 建筑间距
- 民用建筑之间的防火间距：高层与高层≥13m，高层与单多层≥9m
- 厂房之间防火间距按火灾危险性类别确定

### 6.4 安全疏散
- 安全出口数量：每个防火分区≥2个
- 疏散楼梯须设置封闭楼梯间或防烟楼梯间
- 安全出口应分散布置，间距≥5m
- 疏散通道净宽度≥1.1m，疏散门净宽度≥0.9m

### 8.3 消防设施
- 自动喷水灭火系统设置场所
- 室内消火栓系统设置要求
- 火灾自动报警系统设置范围

### 10.1 消防电源与配电
- 一、二类高层建筑须设置双电源或双回路供电
- 消防设备配电线路须采用防火电缆

## 重大隐患判定关联

| 隐患场景 | 隐患级别 |
|---------|---------|
| 安全出口数量不足或被堵塞 | 重大隐患 |
| 消防设施不能正常运行 | 重大隐患 |
| 防火分区被破坏（拆除防火墙） | 重大隐患 |
| 疏散通道净宽度不足 | 重大隐患 |
| 楼梯间未设置封闭/防烟楼梯间 | 重大隐患 |
| 消防电源不符合要求 | 重大隐患 |
