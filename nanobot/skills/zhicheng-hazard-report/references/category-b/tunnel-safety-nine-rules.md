# 隧道施工安全九条规定 — 原文精要

- 源文件: B类规范（国家）\50、隧道施工安全九条规定.docx
- 性质: 部门规章（国家安全生产监督管理总局令）
- 适用范围: 公路、铁路、水利、矿山等隧道工程施工
- 关联清单: references/CASES_XLSX.md → 隧道工程类隐患

## 九条规定（原文要点）

**第一条** 严禁违规穿越突出水、有毒气体等重大风险地层。

**第二条** 严禁擅自改变施工方法，超挖、欠挖。

**第三条** 严禁不进行超前地质预报，盲目施工。

**第四条** 严禁通风不到位，有毒有害气体超限作业。

**第五条** 严禁不按规范进行支护，偷工减料。

**第六条** 严禁特种作业无证上岗。

**第七条** 严禁不按要求配齐安全防护设施。

**第八条** 严禁冒险施救、违章指挥。

**第九条** 严禁瞒报、迟报、漏报安全事故。
