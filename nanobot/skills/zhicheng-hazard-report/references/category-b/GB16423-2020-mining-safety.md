# 金属非金属矿山安全规程 — 原文精要

- 源文件: B类规范（国家）\GB 16423-2020 金属非金属矿山安全规程.pdf
- 性质: 中华人民共和国国家标准
- 编号: GB 16423-2020
- 适用范围: 金属非金属矿山的设计、建设、开采、尾矿库、排土场等
- 关联清单: references/CASES_XLSX.md → 矿山类隐患

---

## 主要章节与要点

### 露天开采
- 最终边坡角须符合设计规定，定期监测边坡稳定性
- 台阶高度、台阶坡面角、机械设备运行安全距离须符合规定
- 排水系统完善，截水沟畅通

### 地下开采
- 通风系统完善，风量、风速满足要求
- 顶板管理：敲帮问顶制度化，及时支护
- 提升运输：钢丝绳定期检查，防跑车装置有效

### 尾矿库
- 滩顶高度、干滩长度、渗流控制满足设计要求
- 排洪系统畅通，在线监测系统运行正常

### 应急救援
- 应急救援预案、应急通讯、避险硐室

## 重大隐患关联
- 边坡失稳/塌方 → 重大隐患
- 通风系统失效/有毒气体超限 → 重大隐患
- 尾矿库超设计坝高/排洪失效 → 重大隐患
