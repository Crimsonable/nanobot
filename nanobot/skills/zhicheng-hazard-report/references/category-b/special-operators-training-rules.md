# 特种作业人员安全技术培训考核管理规定 — 原文精要

- 源文件: B类规范（国家）\45、特种作业人员安全技术培训考核管理规定(2015修正).docx
- 性质: 部门规章（原安监总局令第30号，2015年修正）
- 适用范围: 特种作业人员（电工、焊接与热切割、高处、起重、制冷与空调等）的培训、考核与证书管理
- 关联清单: references/CASES_XLSX.md → 临时用电、动火、高处、起重等

## 核心制度要点
- 培训考核：先培训后考核，考核合格取得《特种作业操作证》方可上岗
- 复审换证：按规定周期复审与换证，逾期未复审证件失效
- 异地作业：跨地区从业的，证件全国范围内有效，接受属地监管
- 考核机构：具备资质并纳入监管，考核过程留痕

## 重大隐患关联
- 特种作业无证上岗 → 重大隐患
- 证件过期未复审继续上岗 → 重大隐患
