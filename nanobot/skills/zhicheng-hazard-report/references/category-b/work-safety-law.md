# 中华人民共和国安全生产法 — 原文精要

- 源文件: B类规范（国家）\安全生产法.pdf
- 性质: 法律（2021修订）
- 适用范围: 在中华人民共和国境内从事生产经营活动的单位的安全生产管理
- 关联清单: references/CASES_XLSX.md → 管理职责、风险分级管控与隐患排查治理、教育培训、特种作业

---

## 核心制度要点
- 主要负责人职责：建立健全并落实安全生产责任制、规章制度和操作规程，保证安全投入（第三条、第二十一条等）
- 双重预防机制：风险分级管控与隐患排查治理制度化、常态化（第十七条）
- 教育培训：从业人员安全教育培训合格后上岗，特种作业必须持证（第二十八条）
- 承包与分包：统一协调管理，明确安全生产职责（第三十八条）
- 重大危险源：登记建档、评估、监控与应急预案（第三十六条）
- 应急救援：应急预案编制与演练、事故信息报告（第四十条及相关条款）
- 法律责任：对主要负责人、生产经营单位及相关人员的处罚加重（第八章）

## 重大隐患关联
- 未建立隐患排查治理制度、未整改重大隐患即组织生产 → 重大隐患
- 未进行安全教育培训、特种作业无证上岗 → 重大隐患
- 重大危险源未监控、无预案、无演练 → 重大隐患
