## 营地建设安全管理强制规定

### 强制性规定要点

**营地选址与评估**
- 营地建设前必须进行安全风险评估，严禁在地质灾害隐患区域、洪水倒灌区域、泥石流通道等危险区域建设营地
- 营地区域应合理规划，办公区、生活区、施工区应分区设置，保持安全防护距离
- 严禁在易燃易爆物品仓库安全距离内设置营地

**房屋建筑安全**
- 营地房屋必须采用符合消防耐火等级要求的材料，严禁使用易燃材料搭建临时建筑
- 宿舍、办公用房等临时建筑与在建工程之间的防火距离应符合GB 50720-2011要求
- 营地用电必须符合临时用电安全规范，严禁私拉乱接

**应急设施**
- 营地必须配备消防设施，设置疏散通道和应急指示标志
- 危险化学品储存须独立设置专用库房，与人员活动区保持安全距离

### 重大隐患判定
- 营地在地质灾害隐患区域或洪水倒灌区域内 → 重大隐患
- 使用易燃材料搭建宿舍/办公用房 → 重大隐患
- 营地与危险化学品仓库安全距离不足 → 重大隐患

## 斜井施工安全管理强制规定

### 强制性规定要点

**设计与方案**
- 斜井施工前必须编制专项施工方案，经审批后方可实施
- 严禁无设计、无方案开挖掘进
- 斜井井口须设置防排水设施，防止地表水倒灌

**开挖与支护**
- 严格按设计开挖，严禁擅自改变开挖方法
- 爆破作业须执行作业票制度，严格执行"一炮三检"和"三人连锁"爆破制度
- 严禁在围岩失稳情况下继续开挖

**通风与气体检测**
- 斜井掘进须配备有效通风设施，风量、风速须满足规范要求
- 必须配备有毒有害气体检测仪器，瓦斯浓度超限须立即撤人
- 严禁无通风措施或通风失效情况下作业

**运输与信号**
- 斜井提升运输须设置完善信号系统，井口设阻车器
- 提升钢丝绳须定期检查，断丝、磨损超限须立即更换
- 严禁超载提升，严禁人货混装

### 重大隐患判定
- 无专项施工方案开挖掘进 → 重大隐患
- 瓦斯浓度超限继续作业 → 重大隐患
- 围岩失稳继续开挖 → 重大隐患
- 提升钢丝绳断丝/磨损超限继续使用 → 重大隐患
- 无阻车器/挡车栏运行 → 重大隐患

## 脚手架安全管理强制规定

### 强制性规定要点

**设计与计算**
- 脚手架工程须编制专项施工方案，超过一定规模的须经专家论证
- 脚手架设计须进行承载力计算，明确搭设参数
- 严禁无设计方案或随意凭经验搭设

**搭设与验收**
- 脚手架构配件须符合国家标准，钢管须有质量合格证明
- 脚手架搭设须按方案进行，严禁擅自更改设计方案
- 脚手架投入使用前须经验收合格，悬挂验收合格标志
- 脚手架拆除须制定拆除方案，由上而下逐层拆除

**使用与检查**
- 脚手架上的施工荷载须符合设计要求，严禁超载
- 使用期间须定期检查，基础沉降、杆件变形、连墙件松动须及时处理
- 六级以上大风须停止高处作业，雨雪后须检查加固

**构造要求**
- 脚手架须设置剪刀撑或斜撑，连墙件须按方案设置
- 作业层须满铺脚手板，设防护栏杆和挡脚板
- 悬挑脚手架须有悬挑支撑结构，严禁悬挑长度超过设计规定

### 重大隐患判定
- 脚手架基础或附着结构不满足承载力要求 → 重大隐患
- 高度超过50m落地脚手架及高度超过20m悬挑脚手架无架体卸荷措施 → 重大隐患
- 连墙件整层缺失 → 重大隐患
- 架体严重变形、杆件缺失 → 重大隐患
- 未经验收投入使用 → 重大隐患
- 脚手架超载使用 → 重大隐患
