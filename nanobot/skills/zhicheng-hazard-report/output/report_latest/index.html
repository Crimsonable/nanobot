高架桥拆除作业区 · 312国道侧翻事故现场

桥梁拆除工程 · 事故处理作业 · 坍塌影响区域

⚠ 含严重隐患
⚠ 3 项严重隐患 🔶 2 项一般隐患
  • 高处作业无防护：作业人员站在数米高桥墩边缘作业，现场未见任何安全带挂靠点、生命绳、防护栏杆或作业平台
    依据《建筑施工高处作业安全技术规范》(JGJ 80-2016) 第4.1.1条（临边作业防护）及第5.1.1条（攀登与悬空作业安全防护）
    严重
  • 结构坍塌风险下违章进入：桥梁已断裂侧翻处于极不稳定状态，仅靠土堆支撑，人员和机械仍在下方紧邻区域作业
    依据《建设工程拆除部分安全技术规范》(JGJ 147-2016) 及《安全生产法》关于危险区域隔离的规定
    严重
  • 机械作业与人员位置冲突：挖掘机斗臂下方和旋转半径内有人员站立，疑似踩踏在机械部件上
    依据《建筑机械使用安全技术规程》(JGJ 33-2012) 第5.1.10条：挖掘机旋转及作业范围内严禁无关人员停留
    严重
  • 个人防护装备佩戴不规范：部分人员疑似未正确佩戴安全帽，在高风险环境下未见穿着高反光背心
    依据《安全帽测试方法及要求》(GB 2811) 及相关防护规定
    一般
  • 现场临时用电与文明施工管理混乱：电缆、气泵管或液压管随意拖地摆放，与机械行驶路径交织，现场堆放大量杂物
    依据《施工现场临时用电安全技术规范》(JGJ 46-2005) 及《建设工程施工现场环境与卫生标准》(JGJ 146-2013)
    一般
